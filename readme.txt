Customize Header & Footer using the Elementor page builder.

== Description ==

The Header & Footer with Elementor plugin allows you to create custom Header and Footer with Elementor for your website.

=== Features ===
- Easily create a custom header/footer layout using Eelementor.
- Users can easily assign the layout to the corresponding header and footer.
- Currently this plugin only compatible with [Ascent](https://wordpress.org/themes/ascent/) theme.
- Many more to come...

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "Header & Footer with Elementor".
3. Click install.
4. Click activate.
